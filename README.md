USVG
====


Scalable UI Components made using Scalable Vector Graphics (SVG) and embedded JavaScript. 


![Design](https://raw.github.com/neonnds/USVG/master/web_ui_preview.png)


Basic Components:
<ul>
	<li>Button</li>
	<li>Button Group</li>
	<li>Switch</li>
	<li>Dial</li>
	<li>Slider</li>
	<li>RadioButton</li>
	<li>CheckBox</li>
	<li>TextField</li>
	<li>TextArea</li>
	<li>Label</li>
	<li>ProgressBar</li>
	<li>DropDownList</li>
 	<li>NumericTextBox</li>
	<li>ComboBox</li>
	<li>ColorPicker</li>
</ul>
